# AlphaHelix

Alpha Version of Helix blockchain protocol based on Arbitrum One

## Isa & Gringo mini game

A lightweight browser mini game lives in `game/`. Open `game/index.html` directly in your browser (or serve the folder with any static file server) to dodge chanclas, grab love hearts, and collect beers that slow down incoming sandals. Mobile-friendly controls and on-screen buttons are included for phones.
